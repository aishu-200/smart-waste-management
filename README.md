🌍 Project Overview
The Smart Waste Management System is a web-based dashboard designed to monitor waste bins across a city. It helps authorities track bin levels, detect when bins are nearly full, and plan efficient garbage collection routes.
This project simulates a smart city waste monitoring system using interactive bins, alerts, and route generation.
🚀 Features
Smart bin monitoring system
Biodegradable & Non-biodegradable bins
Waste level visualization
Automatic alert when bins exceed 80%
Dashboard statistics
Smart garbage collection route generation
Garbage truck movement animation
Filled bins notification system
City location map

📊 System Workflow
Waste is added to bins.
Bin level increases visually.
When bin exceeds 80% capacity, an alert is generated.
The system updates the dashboard.
The garbage collection route is generated.
The truck animation moves across the city map.
